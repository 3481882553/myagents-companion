## 页面清单

| 页面 | 说明 |
|------|------|
| P1 连接页面 | 首次打开，扫码/手动输入连接桌面端 |
| P2 会话列表 | 已连接状态，展示所有会话 |
| P3 聊天阅读器 | 核心页面，渲染 AI 回复 |
| P4 权限审批 | 工具权限弹窗 |
| P5 离线 + Widget | 离线状态和 Widget 渲染 |
| P6 小助理 | 首页卡片入口 + 快捷提问 + 历史会话 |

---

## 交互流程

```
首次打开 App
  │
  ▼
P1 连接页面（扫码 / 手动输入）
  │  配对成功
  ▼
P2 会话列表 ←──────────────────────┐
  │  点击会话                      │
  │  或点击小助理卡片              │
  ├──────────────┐                 │
  ▼              ▼                 │
P3 聊天阅读器   P6 小助理会话       │
  │              │  (标准 Chat)     │  点击通知
  │  收到权限请求 │                 │
  ▼              ▼                 │
P4 权限审批弹窗 ──────────────────┘
  │  审批完成
  └──────────────────────────────┘
```
