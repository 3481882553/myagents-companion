# MVP W7 Mermaid/KaTeX 渲染优化 — 测试文档

> **阶段**：W7 Mermaid/KaTeX 渲染优化  
> **目标**：将 PoC 的 CDN 加载改为 asset:// 本地加载，优化降级策略  
> **测试框架**：Jest + TypeScript

---

## 一、KaTeXBlock 测试

### 1.1 渲染功能

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 行内公式 | `$E=mc^2$` | 正确渲染 |
| 块级公式 | `$$\sum_{i=1}^{n} x_i$$` | 正确渲染 |
| 复杂公式 | 矩阵/积分 | 正确渲染 |
| 渲染失败 | 无效 LaTeX | 降级显示源码 |

### 1.2 动态高度

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| 渲染完成 | postMessage 返回高度 | WebView 高度调整 |
| 高度变化 | 公式内容变化 | 高度动态更新 |

### 1.3 性能

| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 单公式渲染 | 1 个公式 | ≤ 50ms |
| 多公式渲染 | 6 个公式 | ≤ 3s |

## 二、MermaidDiagram 测试

### 2.1 渲染功能

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 流程图 | flowchart | 正确渲染 |
| 时序图 | sequenceDiagram | 正确渲染 |
| 甘特图 | gantt | 正确渲染 |
| 主题切换 | light/dark | 即时生效 |

### 2.2 降级策略

| 用例 | 条件 | 预期结果 |
|------|------|---------|
| 正常渲染 | 节点 ≤ 100 | 显示 SVG |
| 大图降级 | 节点 > 100 | 显示代码文本 |
| 渲染超时 | > 2s | 显示代码文本 |
| 渲染失败 | 无效 Mermaid | 显示代码文本 |

### 2.3 性能

| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 50 节点流程图 | flowchart | ≤ 2s |
| 大型甘特图 | gantt | ≤ 3s |

## 三、Markdown + KaTeX + Mermaid 集成测试

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 纯 Markdown | `# Title` | 正确渲染 |
| Markdown + KaTeX | `# Title\n$E=mc^2$` | 混合渲染 |
| Markdown + Mermaid | `# Title\n```mermaid...``` ` | 混合渲染 |
| 混合内容 | 三者混合 | 全部正确渲染 |
