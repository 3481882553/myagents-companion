# MVP W5 认证 + 连接管理 — 测试文档

> **阶段**：W5 认证 + 连接管理  
> **目标**：验证配对流程、JWT 认证、深度链接、连接状态机  
> **测试框架**：Jest + TypeScript

---

## 一、Auth 服务测试

### 1.1 配对流程

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 正确配对码 | code: '123456' | 返回 JWT Token |
| 错误配对码 | code: '000000' | 抛出错误 |
| 空配对码 | code: '' | 抛出错误 |

### 1.2 Token 管理

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| 存储 Token | `saveToken('token')` | Token 存入 Keychain |
| 读取 Token | `getToken()` | 返回存储的 Token |
| 删除 Token | `removeToken()` | Token 从 Keychain 删除 |
| Token 过期检测 | 检查 Token 过期时间 | 返回 true/false |

### 1.3 Token 续期

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| Token 未过期 | 过期时间 > 24h | 不需要续期 |
| Token 即将过期 | 过期时间 < 24h | 自动续期 |
| Token 已过期 | 过期时间 < 0 | 重新配对 |

## 二、DeepLink Hook 测试

### 2.1 URL 解析

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 有效深度链接 | `myagents://s/ses_abc123?h=192.168.1.5:32101` | `{ sessionId: 'ses_abc123', host: '192.168.1.5:32101' }` |
| 缺少 host | `myagents://s/ses_abc123` | 抛出错误 |
| 缺少 session_id | `myagents://s/?h=192.168.1.5:32101` | 抛出错误 |
| 无效 URL | `invalid://url` | 返回 null |

### 2.2 Host 白名单验证

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 已配对 host | host 在白名单中 | 允许连接 |
| 未配对 host | host 不在白名单中 | 拒绝连接，提示配对 |
| Token 过期 | host 已配对但 Token 过期 | 自动续期或重新配对 |

## 三、ConnectionManager 测试

### 3.1 连接流程

| 用例 | 操作 | 预期状态 |
|------|------|---------|
| 初始状态 | 创建 ConnectionManager | disconnected |
| 开始连接 | `connect('192.168.1.5:32101')` | connecting |
| 连接成功 | 服务端返回 200 | connected |
| 连接失败 | 服务端返回错误 | disconnected |

### 3.2 自动重连

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| 指数退避 | 连续失败 | 500ms → 1s → 2s → ... → 30s |
| 重连成功 | 成功连接后断开 | 从 500ms 重新开始 |
| 最大重试 | 超过 10 次 | 停止重连 |

### 3.3 网络变化处理

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| WiFi → 4G | 网络切换 | 主动断开并重连 |
| 飞行模式开 | 网络断开 | 进入重连状态 |
| 飞行模式关 | 网络恢复 | 自动重连 |

## 四、ConnectionScreen 测试

### 4.1 UI 渲染

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| 初始状态 | 打开连接页 | 显示手动输入表单 |

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| 输入 IP + 端口 | 填写表单 | 输入框显示内容 |

| 连接失败 | 服务端返回错误 | 显示错误信息 |
