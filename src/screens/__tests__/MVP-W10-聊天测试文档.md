# MVP W10 深度链接 + IM 集成 — 测试文档

> **阶段**：W10 深度链接 + IM 集成  
> **目标**：实现 ChatScreen + SessionListScreen + 深度链接 + 消息发送  
> **测试框架**：Jest + TypeScript

---

## 一、SessionListScreen 测试

### 1.1 渲染

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 空列表 | sessions: [] | 显示空状态 |
| 有会话 | sessions: [1,2,3] | 显示会话卡片列表 |
| 点击会话 | 点击卡片 | 跳转到 ChatScreen |

### 1.2 会话卡片

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 显示标题 | title: '测试' | 显示标题 |
| 显示最后消息 | lastMessage: 'Hello' | 显示预览 |
| 显示时间 | lastMessageAt: Date.now() | 显示"刚刚" |

## 二、ChatScreen 测试

### 2.1 渲染

| 用例 | 输入 | 预期输出 |
|------|------|---------|
| 消息列表 | messages: [...] | 渲染消息列表 |
| 空消息 | messages: [] | 显示空状态 |
| 用户消息 | role: 'user' | 右对齐气泡 |
| AI 消息 | role: 'assistant' | 左对齐无气泡 |

### 2.2 消息发送

| 用例 | 操作 | 预期结果 |
|------|------|---------|
| 输入文本 | 输入 'Hello' | 输入框显示内容 |
| 点击发送 | 发送按钮 | 调用 API 发送 |
| 发送后清空 | 发送成功 | 输入框清空 |
| 空消息不发送 | 输入为空 | 按钮禁用 |

### 2.3 SSE 流式渲染

| 用例 | 事件 | 预期结果 |
|------|------|---------|
| 收到 chunk | chat:message-chunk | 追加到当前消息 |
| 收到 complete | chat:message-complete | 完成当前消息 |
| 收到 thinking | chat:thinking-start | 显示思考动画 |
